Расширенное описание функционала кнопок для сервиса организации закупокНиже приведено детальное описание функционала всех кнопок, которые будут использоваться в сервисе. Описание разделено по ролям пользователей: покупатель, организатор и поставщик.1. Функционал кнопок для покупателя1.1. Личный кабинет покупателяСоздать запрос:Описание: Позволяет покупателю создать новый запрос на товар.Функционал:Открывает форму с полями: название товара, количество, город получения, примечание.После заполнения формы запрос добавляется в общий чат.Результат: Запрос отображается в вертикальной ленте чата.Поиск товаров:Описание: Позволяет покупателю искать товары по названию.Функционал:Открывает строку поиска.При вводе названия товара и нажатии "Искать" происходит переход в чат, где упоминаются соответствующие товары.Результат: Отображение результатов поиска в чате.Мои запросы:Описание: Отображает список запросов, созданных покупателем.Функционал:Показывает запросы в виде списка с количеством новых заявок по каждому товару.При нажатии на запрос происходит переход в чат, где упоминаются соответствующие закупки.Результат: Переход в чат с закупками по выбранному товару.Мои закупки:Описание: Отображает список закупок, в которых участвует покупатель.Функционал:Показывает закупки в виде списка с указанием активности (например, необходимость подтверждения участия).При нажатии на закупку происходит переход в чат с деталями закупки.Кнопка "Удалить из заявки" позволяет выйти из закупки.Результат: Переход в чат закупки или удаление из заявки.Мои приглашения и сообщения:Описание: Отображает приглашения в закупки и сообщения от других пользователей.Функционал:Показывает список приглашений и сообщений.При нажатии на приглашение происходит переход в чат закупки.При нажатии на сообщение открывается окно для ответа.Результат: Переход в чат или ответ на сообщение.История закупок:Описание: Отображает историю закупок, в которых участвовал покупатель.Функционал:Показывает список завершенных закупок с указанием общей суммы и статуса.Результат: Просмотр истории закупок.Пополнить баланс / Вывести средства:Описание: Управление балансом покупателя.Функционал:"Пополнить баланс" — открывает страницу оплаты через API банка."Вывести средства" — открывает форму для вывода средств на указанный счет.Результат: Изменение баланса покупателя.2. Функционал кнопок для организатора2.1. Личный кабинет организатораСоздать закупку:Описание: Позволяет организатору создать новую закупку.Функционал:Открывает форму с полями: название товара, единица измерения, город получения, описание, размер комиссии (1-4%).После заполнения формы закупка добавляется в горизонтальный слайдер.Результат: Закупка отображается в слайдере и чате.Открытые закупки:Описание: Отображает список активных закупок, созданных организатором.Функционал:Показывает закупки в виде списка.При нажатии на закупку происходит переход в чат с деталями закупки.Результат: Переход в чат закупки.Закупки в стадии оплаты:Описание: Отображает закупки, которые находятся в стадии оплаты.Функционал:Показывает список закупок с указанием даты остановки закупки.При нажатии на закупку происходит переход в закрытый чат участников.Результат: Переход в закрытый чат.История закупок:Описание: Отображает историю завершенных закупок.Функционал:Показывает список закупок с указанием общей суммы и вознаграждения организатора.Результат: Просмотр истории закупок.Создать новость:Описание: Позволяет организатору публиковать новости.Функционал:Открывает форму для создания новости (заголовок, содержание).После публикации новость отображается в ленте новостей.Результат: Новость публикуется в ленте.2.2. Кнопки в слайдере (видимые только организатору)Стоп-сумма:Описание: Останавливает закупку и создает закрытый чат для участников.Функционал:После нажатия участники получают уведомление с просьбой подтвердить участие.Участники, подтвердившие участие, переносятся в закрытый чат.Результат: Создание закрытого чата.Утвердить поставщика:Описание: Утверждает выбранного поставщика.Функционал:После нажатия поставщик добавляется в информацию о закупке.Результат: Поставщик утвержден.Создать и отправить таблицу чеков:Описание: Формирует сводную таблицу с чеками покупателей.Функционал:После нажатия создается таблица с данными о платежах.Таблица отправляется поставщику.Результат: Поставщик получает таблицу.Закрыть закупку:Описание: Завершает закупку.Функционал:После нажатия закупка перемещается в историю.Результат: Закупка завершена.3. Функционал кнопок для поставщика3.1. Личный кабинет поставщикаСоздать карту компании:Описание: Позволяет поставщику создать карту компании.Функционал:Открывает форму с полями: название компании, адрес, ИНН, контактный телефон, электронная почта.Результат: Карта компании создана.Загрузить прайс-лист:Описание: Позволяет поставщику загрузить прайс-лист.Функционал:Открывает форму для загрузки файла.После загрузки прайс-лист отображается в публичной части кабинета.Результат: Прайс-лист загружен.Таблица заказов:Описание: Отображает сводные таблицы с заказами.Функционал:Показывает таблицы с данными о заказах.Кнопка "Отправить закрывающие документы" позволяет отправить документы покупателям.Результат: Документы отправлены.История отгрузок:Описание: Отображает историю отгрузок.Функционал:Показывает список завершенных отгрузок.Результат: Просмотр истории.Написать на страницу новостей:Описание: Позволяет поставщику публиковать новости.Функционал:Открывает форму для создания новости.Результат: Новость опубликована.
